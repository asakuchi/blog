# dc コマンド

desk calculator 電卓
らしい。

逆ポーランド記法で計算を行う。

# 使い方

```bash
dc -e '2 3 * p'
6
```

# 演算子とか

表示 (`p`)

```
dc -e '100 p'
100
```

標準入力 (`?`)

```dc
echo 200 | dc -e '? p'
200
```

足し算 (`+`)

```
dc -e '10 20 + p'
30
```

引き算 (`-`)


```
dc -e '10 3 - p'
7
```

掛け算 (`*`)

```
dc -e '12 5 * p'
60
```

割り算 (`/`)

```
dc -e '20 3 / p'
6
```

剰余（あまり） (`%`)

```
dc -e '20 3 % p'
2
```

累乗 (`^`)

```
dc -e '2 10 ^ p'
1024
```

累乗の剰余 ('|')

3^4%5 = 81%5 = 1

```
dc -e '3 4 5 | p'
1
```

平方根 (`v`)

```
dc -e '64 v p'
8
```

商と余りを同時計算 (`~`)

```
dc -e '20 3 ~ f'
```

```
2
6
```
